Uwaga:

  • Jeśli używasz zwykłego linku .../edit, Google zwykle blokuje osadzanie w iframe (nagłówek X-Frame-Options).
  • Aby osadzić arkusz, w Google Sheets wybierz: File → Publish to the web → Embed i skopiuj link zawierający /pubhtml.
  • Jeśli arkusz nie jest udostępniony publicznie (Anyone with the link), kliknięcie linku otworzy stronę wymagającą logowania/zgody.

Kilka szybkich wskazówek:
  1. Jeśli po kliknięciu widzisz stronę „Prośba o dostęp”, otwórz arkusz w Google i ustaw udostępnianie na: Anyone with the link — Viewer.
  2. Aby osadzić arkusz bez problemów w iframe użyj opcji Publish to the web → Embed (kopiuj link /pubhtml).
